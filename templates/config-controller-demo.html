Rtcomm Config


{
	"server" : "hostname",
	"port": 1883,
	"rtcommTopicPath": "/rtcomm",
	"broadcastAudio": true,
	"createEndpoint": true,
	"userid" :'',
	"presenceTopic": "demo"

}

A JSON configuration file such as the example above is needed to establish the communciation with the MQTT server. This file needs to go on the end of the HTML file, before the </body> tag. The configuration file accepts the following parameters:

Name Data Type Description
server string The hostname or ip address of the server with the MQTT broker
port int The designated port on the server
rtcommTopicPath string MQTT Path that the client will be subscribed to
createEndpoint boolean If an endpoint should be created automatically
userid string User identifier 
		


<div ng-controller="RtcommConfigController" ng-init="init('rtcommConfig.json')" ></div>
